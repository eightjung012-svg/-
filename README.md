## ✨ 주요 기능
1. **메인 페이지**: 최신 영화 및 OTT 시리즈 포스터 그리드 뷰 제공
2. **리뷰 작성 기능**: 별점 평가 및 한줄평 등록 (실시간 DOM 조작)
3. **데이터 저장**: LocalStorage를 활용하여 새로고침 후에도 리뷰 유지
4. **반응형 웹**: 데스크톱 및 모바일 환경 완벽 지원
5. **실시간 영화 검색 (Real-time Search)**
* 상단 검색바를 통해 원하는 영화나 드라마를 즉시 찾을 수 있음
* 키워드 입력 시 화면이 **자동으로 필터링**되어 검색 결과를 보여줌
6. **SPA 방식의 메뉴 이동 (Navigation)**
* **[홈] - [박스오피스] - [내 리뷰]** 메뉴 탭을 제공.
* 페이지 새로고침 없이 부드럽게 화면이 전환되어 사용자 경험(UX)을 극대화.
